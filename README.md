📦 AtliQ Mart Supply Chain Analysis
📊 Project Overview

The AtliQ Mart Supply Chain Analysis Dashboard is a Power BI project developed to analyze supply chain performance across different cities, customers, and products. The dashboard helps monitor delivery efficiency, order fulfillment, and product availability to support better operational decision-making.

This project focuses on key supply chain KPIs such as On-Time Delivery, In-Full Delivery, OTIF, Line Fill Rate, and Volume Fill Rate, providing insights into delivery performance and operational efficiency.

📈 Key Performance Indicators

OT % (On-Time Delivery) – Measures how often orders are delivered on time.

IF % (In-Full Delivery) – Measures whether orders are delivered in the correct quantity.

OTIF % – Indicates orders delivered both on time and in full.

LIFR % (Line Item Fill Rate) – Percentage of ordered items successfully fulfilled.

VOFR % (Volume Fill Rate) – Percentage of ordered volume successfully delivered.

These metrics help evaluate the reliability and efficiency of the supply chain operations.

📊 Dashboard Sections
1️⃣ Overview

The overview page provides a high-level view of supply chain performance.

Key insights include:

Comparison of delivery KPIs against target values

City-wise delivery performance

Overall fulfillment efficiency across locations

2️⃣ Customer & Product Analysis

This section analyzes order fulfillment performance at the customer and product level.

Key insights include:

Order distribution by delivery status

Category-wise order analysis

Customer-level delivery performance metrics

Product-level fulfillment performance

3️⃣ Trend Analysis

This page focuses on monitoring KPI performance trends over time.

Key insights include:

Trends for OT %, IF %, and OTIF %

Performance comparison against target values

Identification of delivery performance patterns

4️⃣ Insights & Solutions

This section highlights key operational insights that can help improve supply chain performance.

Example areas analyzed include:

Delivery delays

Product fulfillment issues

Customer-level performance gaps

Opportunities to improve delivery efficiency
